tags: help

Step1. Set idf file before running simulation

Once you set all the setting for energy simulation, set following setting to the idf file. 1. In Output:Table:SummaryReports chapter, set Report 1 Name to AllSummary ({filename}/images/instruction1.png) 2. In OutputControl:Table:Style chapter, set Column Separator to CommaAndHTML and Unit Conversion to JtoKWH ({filename}/images/instruction2.png)

Step2. Run the simulation

Step3. Upload the simulation result

Once you finished the simulation, go to the upload page[link] and input all the information and upload html file from the simulation result. ({filename}/images/instruction3.png)
